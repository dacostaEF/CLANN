# Assets

Este diretório deve conter os seguintes arquivos de assets:

- `icon.png` - Ícone do app (1024x1024)
- `splash.png` - Imagem de splash screen
- `adaptive-icon.png` - Ícone adaptativo para Android (1024x1024)
- `favicon.png` - Favicon para web

Estes arquivos são referenciados no `app.json` e são necessários para o build do app.
